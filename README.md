<p align="center"><b>GO project with mysql and nginx reverse proxy on Docker.</b></p>

<br>

## start
docker-compose up -d
## stop
docker-compose stop
## delete
docker-compose down
## check docker status
docker-compose ps
## execute command in GO
docker-compose exec go sh
## get out of node
exit
## execute GO project
After `execute command in GO` process
go run main.go
## install GO module
After `execute command in GO` process
go get xxxxxxxxxx
